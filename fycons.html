Fycons

Your friendly neighborhood icon-font library

Usage

Fycons are generally tracked and vertically aligned for you. When used in parent elements such as buttons, they should correctly display and scale with inherited font-sizes. The advised tag to use for fycons is <span class="fycon-*" aria-hidden="true"></span>.

The attribute aria-hidden="true" prevents some screenreaders from announcing the character code (reference).

Note: class names shown below under "List of Fycons" need to be prefixed with fycon-. For example, <span class="fycon-action-etc" aria-hidden="true"></span>.

Add a Plus ("+")

[Livefyre Bootstrap 2.x only] Use the .add-plus class to add a small "+" next to the Fycon.

Create a New Twitter Rule

<!-- Livefyre Bootstrap 2.x only -->
<a href="#"><span class="fycon-source-twitter add-plus" aria-hidden="true"></span> Create a New Twitter Rule</a>
<button class="btn btn-default"><span class="fycon-folder add-plus" aria-hidden="true"></span> Add a Folder</button>

In buttons

Add the fycon tag inside of a container element. Do not add the fycon class to container itself.

<!-- If you're using Livefyre Bootstrap 2.x, skip the "lf-" prefixes. -->

<button class="lf-btn lf-btn-primary">
  <span class="fycon-plus" aria-hidden="true"></span> New Collection
</button>

<button class="lf-btn lf-btn-default">
  <span class="fycon-action-like" aria-hidden="true"></span> Like
</button>

<button class="lf-btn lf-btn-link">
  <span class="fycon-action-reply" aria-hidden="true"></span> Reply
</button>

Colored Fycons

Social media fycons ship with colored variants. Simply add the .colored class and the correct brand color will be applied to it.

<span class="fycon-source-livefyre colored" aria-hidden="true"></span>
<span class="fycon-source-facebook colored" aria-hidden="true"></span>
<span class="fycon-source-twitter colored" aria-hidden="true"></span>
...
