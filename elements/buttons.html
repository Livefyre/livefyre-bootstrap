Buttons

Types

Default Button

Colors

We have a number of button colors available that are used throughout our designs. Semantically, the class names are defined by action and not color.

The exception is the new Hollow button. It's semi-transparent so it looks slick on top of a photo.

Disabled Buttons

Sizes

There are 3 additional sizes as well as a full-width block-level button size. These are designed to match input and form control heights.

Fycons in Buttons

Sizes

Fycon sizes will adjust based on the button size.

However, you can further fine tune the size of fycons in buttons using .fycon-xs, .fycon-sm, .fycon-lg, .fycon-xl and .fycon-inherit (uses the same font size as the button).

Use these classes in the fycon's span container.

Add a Plus ("+")

Use the .add-plus class to add a small "+" next to the Fycon. Works with different fycon sizes.
