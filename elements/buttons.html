What's New in v2.x

  • New semi-transparent .btn-hollow style.
  • Attach a plus symbol next to a fycon using the .add-plus class.
  • Improved ability to fine tune fycon sizes.
  • Improved vertical centering.
  • Constructed using new button height variables (@btn-height-lg: 46px) instead of vertical padding to ensure better button height consistency.
  • All buttons now have the same border-radius (2px).

Fycons in Buttons

Sizes

Fycon sizes will adjust based on the button size.

However, you can further fine tune the size of fycons in buttons using .fycon-xs, .fycon-sm, .fycon-lg, .fycon-xl and .fycon-inherit (uses the same font size as the button).

Use these classes in the fycon's span container.

{% include btn-toggle-code.html target="buttons-fycons" %}
{% highlight html %} {% endhighlight %}
{% highlight html %}var Button = require('react-bootstrap').Button; var Fycon = require('../Fycon'); // Custom LF component - Put the font name in the "ico" attribute, leaving out the "fycon-" part of the class name. {% endhighlight %}
