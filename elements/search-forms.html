Search Forms

a.k.a. Query Inputs

Usage

This is a custom Livefyre component. The markup comes from our React component, Simple.js, found in asset-manager and react-components.

The component's width is determined by its container. Each example below is wrapped in a <div class="col-md-6">.

Default

{% include btn-toggle-code.html target="search-forms-default" %}
{% highlight html %}
{% endhighlight %}
{% highlight html %}var QueryInput = require('../text-input/Simple'); {% endhighlight %}

With Loader

This is what it looks like with the loader rendered by Asset Manager's Simple.js. Note the addition of a <div class="input-group"> wrapper.

{% include btn-toggle-code.html target="search-forms-with-loader" %}
{% highlight html %}
{% endhighlight %}
{% highlight html %}var QueryInput = require('../text-input/Simple'); {% endhighlight %}

With Button

When using a submit button, add the .query-input-with-button class. It removes the magnifying class fycon from the text field. Note the addition of a <div class="input-group"> wrapper.

{% include btn-toggle-code.html target="search-forms-with-button" %}
{% highlight html %}
{% endhighlight %}
{% highlight html %}var Button = require('react-bootstrap').Button; var Fycon = require('../Fycon'); var QueryInput = require('../text-input/Simple'); {% endhighlight %}

Sizes

Search forms come in default and large sizes.

For the large form, add the .query-input-lg class to the container and .input-lg and .btn-lg to their respective elements.
