What's New in v2.x

  • All gray values have changed. Make sure your text is still readable, backgrounds aren't too dark/light, etc.
  • New variables @gray-darkest, @gray-darkish and @gray-lightest.
  • Using hex codes for all (instead of tint) so it's easier to match a variable name with what you find in the inspector.
