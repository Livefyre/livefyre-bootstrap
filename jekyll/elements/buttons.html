Fycons in Buttons

Sizes

You can fine tune the size of fycons in buttons using .fycon-xs, .fycon-sm, .fycon-lg, .fycon-xl and .fycon-inherit (uses the same font size as the button).

Use these classes in the fycon's span container.

Fycon sizes will adjust based on the button size.

{% highlight html %} {% endhighlight %}

Add a Plus ("+")

Use the .add-plus class to add a small "+" next to the Fycon. Works with different fycon sizes.
